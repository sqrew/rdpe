RDPE - Reaction Diffusion Particle Engine

Examples

RDPE includes many self-documented examples in the examples/ directory. Each example contains detailed comments explaining the concepts it demonstrates.

Running Examples

cargo run --example <name>

Available Examples

ExampleDescription
getting_startedMinimal RDPE simulation - start here!
bouncingParticles bouncing off walls
fountainCone emitter with gravity and lifecycle
boidsClassic flocking algorithm with spatial hashing
predator_preyChase and evade behaviors between types
infectionSIR model with Convert rule
particle_lifeEmergent behavior from interaction matrix
attractorMouse-controlled attraction force
orbitTime-animated orbiting attractor
noisyFlow field using built-in noise functions
swirlCustom WGSL functions for vortex effect
explosionBurst emitter for one-shot effects
rainBox emitter for environmental particles
rocketDownward cone emitter for thrust
sphere_shellSphere emitter for 3D distributions

Learning Path

  1. Start with basics: getting_started, bouncing, fountain
  2. Explore interactions: boids, predator_prey, infection, particle_life
  3. Custom shaders: attractor, orbit, noisy, swirl
  4. Emitter types: explosion, rain, rocket, sphere_shell

Each example file contains //! doc comments explaining what it demonstrates and suggestions for experimentation.
